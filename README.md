# Binly Citizen App

This is the React Native mobile application for citizens using Binly, an AI-powered credit-based waste management platform. Citizens can scan waste, get AI-based segregation suggestions, earn credits for correct disposal, and redeem rewards via UPI during the SRM campus pilot.

## Features (Planned)

- User registration and login
- Camera-based waste scan (TFLite classification)
- Composting tips using Gemma-2B
- Credit wallet and transaction history
- QR code for collectors to scan
- Razorpay UPI redemption

## Tech Stack

- React Native (Expo or bare)
- TypeScript or JavaScript
- REST API calls to FastAPI backend
